## 🤝 如何贡献 | How to Contribute

本项目欢迎以下形式的贡献：

We welcome the following types of contributions:

### 1. 🌍 翻译 | Translations

如果你会其他语言，欢迎在 `translations/` 目录下添加对应语言的声明文件。

If you speak other languages, feel free to add a declaration file in the `translations/` directory.

文件命名格式：`translations/README.[语言代码].md`

例如：
- `translations/README.ja.md`（日语）
- `translations/README.ko.md`（韩语）
- `translations/README.fr.md`（法语）

### 2. 💡 建议与讨论 | Suggestions & Discussions

- 对「通宝」这一命名有任何想法，欢迎开 [Issue](https://github.com/tongbao2/tongbao/issues) 讨论
- 发现错别字或表述不准确，欢迎提 PR 修正

### 3. 📣 传播推广 | Spread the Word

- 在你的项目、文章、演讲中使用「通宝」
- 在社交媒体上分享本项目
- 向你的朋友和同事介绍「通宝」

## 📝 提交规范 | Commit Convention

提交信息请使用以下格式：

```
<type>: <description>

类型说明：
- docs: 文档修改
- fix: 修正错误
- feat: 新增内容
- i18n: 翻译相关
```

示例：
```
docs: 修正历史朝代表格中的错别字
i18n: 添加日语翻译文件
feat: 新增适用场景说明
```
