## 📖 名称释义 | Name Explanation

| 字 | 含义 |
|----|------|
| 通 (Tōng) | 流通、通用、互通 — Circulation, universal, interconnected |
| 宝 (Bǎo) | 宝贝、珍贵、财富 — Treasure, precious, wealth |

**通宝** 自古是中国货币的传统称谓（如"开元通宝"、"康熙通宝"），寓意流通之宝，与现代 Token 的概念高度契合。

"通宝" has been a traditional Chinese monetary term since ancient times (e.g., "开元通宝", "康熙通宝"), meaning "circulating treasure" — perfectly aligned with the modern concept of Token.
