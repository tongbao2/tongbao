## 📖 名称释义 | Name Explanation

| 字 | 拼音 | 含义 |
|----|------|------|
| 通 | Tōng | 流通、通用、互通 — Circulation, universal, interconnected |
| 宝 | Bǎo | 宝贝、珍贵、财富 — Treasure, precious, wealth |

**通宝** 自古是中国货币的传统称谓，历朝历代皆有沿用：

| 朝代 | 货币名称 |
|------|---------|
| 唐朝 | 开元通宝 |
| 宋朝 | 宋元通宝 |
| 明朝 | 洪武通宝 |
| 清朝 | 康熙通宝 |

> 「通宝」寓意**流通之宝**，与现代 Token（可流通的数字凭证）的概念高度契合，是跨越千年的完美命名。
>
> "通宝" has been a traditional Chinese monetary term for over a millennium, meaning **"circulating treasure"** — perfectly aligned with the modern concept of Token as a transferable digital credential.

## 🆚 为什么选「通宝」？| Why "通宝"?

| 译名 | 问题 |
|------|------|
| 令牌 | 偏技术感，不够大众化，难以普及 |
| 代币 | 带有"虚拟货币"负面联想，语义狭窄 |
| 凭证 | 过于书面，缺乏流通感与文化温度 |
| 通证 | 生造词，无历史根基，读来生硬 |
| **通宝** ✅ | 历史悠久、朗朗上口、寓意精准、文化认同感强 |
